# あ！デザイン — デザイン思考への扉

> デザインとは、物事をより良くするための思考です。身の回りのものすべてを観察することから始まり、日常に隠された美と知恵を発見する旅です。

NHK教育テレビ『デザインあ』のデザイン知識を整理したインタラクティブな単一ページウェブサイトです。観察・分解・配置・タイポグラフィから実践まで、デザイン思考を体系的に学べます。

🌐 **デモサイト：** [hisunfei.github.io/Ah-design](https://hisunfei.github.io/Ah-design/)

---

## ✨ 技術的特長

- **単一ファイル**：HTML + CSS + JS 一体化、外部依存ゼロ
- **豊富なインタラクション**：分解アニメーション、ドラッグ＆ドロップ、キャンバス描画、紋様作成、フォントウェイトスライダー
- **レスポンシブデザイン**：スマホ・タブレット・デスクトップで快適に閲覧可能
- **スクロールアニメーション**：スクロールに合わせてコンテンツがフェードイン
- **イースターエッグ**：トップページの「あ」をタップするとカラフルに回転 🎨

## 🚀 ローカルで実行

```bash
# リポジトリをクローン
git clone https://github.com/hisunfei/Ah-design.git
cd Ah-design

# ローカルサーバーを起動（いずれか一つ）
python3 -m http.server 8080
# または
npx serve .
```

その後、`http://localhost:8080` にアクセス
