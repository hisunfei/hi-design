# Ah! Design — Design Thinking for Everyone

> Design is the act of thinking about how to make things better. It starts with observing the world around you and discovering the beauty and wisdom hidden in everyday life.

An interactive single-page website that organizes design knowledge from NHK Educational TV's *Design Ah!* — guiding you through observation, disassembly, arrangement, typography, and hands-on practice to build a systematic understanding of design thinking.

🌐 **Live Demo:** [hisunfei.github.io/Ah-design](https://hisunfei.github.io/Ah-design/)

---

## 📖 Sections

| # | Section | Description |
|---|---------|-------------|
| 01 | 🔍 Observation | Design wisdom in 6 everyday objects (egg, teacup, key, paperclip, zipper, mandarin orange) + observation methodology |
| 02 | 🔧 Disassembly | Interactive teardown animations: pencil / bento box / book — tap to explode into parts, tap again to reassemble |
| 03 | 🧩 Let's Collect! | Interactive sorting: 24 elements arranged by color / size / type — discover the beauty of patterns |
| 04 | 🔤 Typography | Font-weight slider + letter-spacing comparison + "Typographic Song" knowledge |
| 05 | 📐 Layout Design | Good vs. bad layout comparisons + the four principles: alignment, contrast, repetition, proximity |
| 06 | 🔶 Patterns | 48-cell interactive pattern canvas (click to rotate) + patterns from Chinese, Japanese, Greek, and Islamic cultures |
| 07 | 🏗️ Spatial Design | 6 architectural design concepts: symmetry, nature, light, circulation, scale, in-between space |
| 08 | 📖 10 Design Principles | Core design principles distilled from the show |
| 09 | 👨🎨 Words from Design Masters | Design philosophy from 12 masters worldwide (Kengo Kuma, Kenya Hara, Dieter Rams, Jony Ive, Wang Shu…) |
| 10 | ✏️ Hands-On Practice | Free drawing canvas + 4 everyday design challenges |

---

## ✨ Technical Highlights

- **Single file**: HTML + CSS + JS all-in-one, zero external dependencies
- **Rich interactions**: disassembly animations, drag-and-drop sorting, canvas drawing, pattern creation, font-weight sliders
- **Responsive design**: works seamlessly on phones, tablets, and desktops
- **Scroll animations**: content fades in on scroll for a smooth reading experience
- **Easter egg**: tap the "Ah" on the homepage for a colorful spin 🎨

## 🚀 Run Locally

```bash
# Clone the repo
git clone https://github.com/hisunfei/Ah-design.git
cd Ah-design

# Start a local server (pick one)
python3 -m http.server 8080
# or
npx serve .
```

Then visit `http://localhost:8080`
